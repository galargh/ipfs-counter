ipfs-counter
=======================

> Counting the server nodes in the IPFS DHT

Crawls the IPFS DHT to give some metrics. Note: this project is a WIP and resulting stats should not be taken as definitive.

## Documentation

### Install

`go install`

### Build

`go build`
